Start and setup machine
	Press on
	Move machine near origin, inside bound
	Press set origin and wait until origin is set

Manual and configure for the first foil
	Insert reference foil, press clamp
	Press step to move to next step
	Change X, Y value accordingly then press Apply to apply modifier value
	...
	Press clamp again to unclamp and take the foil out
	Press step to move to next step
	Now the program is ready

Auto run
	Press clamp
	Press Run and wait for it
	Press clamp again to unclamp and take the foid out
	Wait for it to run back to origin again

Auto program step
	From origin
	Move x8369 y0 lt0 nb0
	Move to y1 ys1
	Move to y2 ys2
	Move to y1 ys2
	Move to x2 xs2
	Move to y2 ys2
	Move to y1 ys2

Check and tune
	Final stage has tune parameter (delta deep, delta left, delta right)
